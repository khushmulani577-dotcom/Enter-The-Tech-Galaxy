Enter the Tech Galaxy

By Jiya Mulani, Class 8C, Delhi World Public School

Welcome to the Tech Galaxy! Here, we explore the wonders of technology, from coding to innovation. Dive into the stars of knowledge and let your imagination soar.

About the Project

This project, "Enter the Tech Galaxy," is my entry for the Techno Week competition at Delhi World Public School. As a student in Class 8C, I wanted to create something that combines creativity, technology, and a touch of the future. The website features a futuristic design with a galaxy background, twinkling stars, and smooth animations to make learning about tech fun and engaging.

I learned how to use HTML, CSS, and JavaScript to build interactive web pages, and even incorporated a simple Node.js server to host it. This project represents my journey into the world of web development and my passion for technology. It includes responsive design to work on all devices, ensuring accessibility for everyone.

The theme is inspired by space and galaxies, symbolizing the vast possibilities in tech. I chose colors like deep space blue, light purple, and glowing white to evoke a sense of wonder and innovation.

My Project Journey

Building this website was an exciting adventure! I started by researching futuristic themes and color palettes. Then, I designed the layout to be clean and responsive, ensuring it looks great on both computers and mobile devices. I added animations like fading text and twinkling stars to bring the galaxy to life.

One of the challenges was making the background move gently without distracting from the content. I also included an interactive button that reveals more information, making the site more engaging. Finally, I tested it on different devices to ensure it's mobile-friendly.

Key steps in my journey:

  • Planning the design and gathering inspiration from space-themed websites.
  • Learning HTML for structure, CSS for styling, and JavaScript for interactivity.
  • Implementing the Node.js server to serve the page locally.
  • Adding smooth transitions and ensuring the page loads quickly.
  • Iterating based on feedback to make it perfect for the competition.

Technologies Used

In this project, I utilized several technologies to create a modern, interactive website:

  • HTML: For the basic structure of the webpage, including headings, sections, and buttons.
  • CSS: For styling, including the galaxy background, animations, and responsive design. I used Google Fonts like Orbitron for headings and Poppins for body text.
  • JavaScript: To add interactivity, such as the button that toggles hidden content.
  • Node.js: With Express.js to create a simple server that hosts the website and logs "Tech Galaxy Server Running" in the console.

These tools allowed me to build a professional-looking site that showcases my skills in web development.

Inspiration and Goals

My inspiration came from the endless universe and how technology connects us to new worlds. I aimed to create a website that not only looks futuristic but also educates visitors about tech in a fun way. For Techno Week, I wanted to stand out by combining art and code.

Goals achieved:

  • Create a clean, modern design that's easy to navigate.
  • Make it fully responsive for mobile and desktop.
  • Incorporate subtle animations to enhance user experience without overwhelming the content.
  • Include personal touches, like my name and school, to make it unique.

This project has boosted my confidence in coding and sparked my interest in pursuing more advanced tech projects in the future.

Future Plans

After Techno Week, I plan to expand this project by adding more interactive features, like a quiz on tech topics or a virtual tour of the galaxy. I also want to learn more about advanced web technologies, such as React for dynamic interfaces or backend databases to store user data.

Additionally, I might integrate APIs to fetch real-time data, like space facts or tech news, to make the site even more engaging. My ultimate goal is to inspire other students to explore technology and creativity.
